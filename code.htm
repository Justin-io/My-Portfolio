GitHub Setup Guide

Step-by-step instructions to create your portfolio and deploy it

Creating Your GitHub Portfolio

1

Create a GitHub Account

Go to github.com and sign up for a free account. Use your email address and create a strong password.

 Username: your-username
 Password: 
 Email: your-email@example.com
2

Create a New Repository

Click the "+" icon in the top right corner and select "New repository". Name it my-portfolio and make it public.

Repository name: my-portfolio
Description: My coding portfolio
 Public
 Initialize with README
3

Upload Your Files

Click "uploading an existing file" and drag your portfolio files (index.html, style.css, script.js) to the upload area.

 index.html
 style.css
 script.js
 images/
4

Enable GitHub Pages

Go to your repository settings, scroll down to "GitHub Pages", and select the main branch as the source.

Settings  Pages  Source: main branch  Save
5

Visit Your Live Site

After a few minutes, your site will be live at: https://[yourname].github.io/my-portfolio

 Your site is live!
 https://yourname.github.io/my-portfolio

Portfolio Template

Here's a basic HTML template to get you started:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Portfolio</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <header>
        <h1>Your Name</h1>
        <p>Web Developer</p>
    </header>
    
    <main>
        <section id="about">
            <h2>About Me</h2>
            <p>Your bio goes here...</p>
        </section>
        
        <section id="projects">
            <h2>My Projects</h2>
            <div class="project">
                <h3>Project 1</h3>
                <p>Description of your project...</p>
            </div>
        </section>
        
        <section id="contact">
            <h2>Contact Me</h2>
            <form>
                <input type="text" placeholder="Your Name">
                <input type="email" placeholder="Your Email">
                <textarea placeholder="Your Message"></textarea>
                <button type="submit">Send</button>
            </form>
        </section>
    </main>
    
    <footer>
        <p>© 2025 Your Name</p>
    </footer>
    
    <script src="script.js"></script>
</body>
</html>
