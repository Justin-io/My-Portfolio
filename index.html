Q-SAFE: Quantum-Secure Application Framework for Education

Framework Development

Q-SAFE is a security-by-design framework for educational and grassroots technology platforms, integrating offensive security testing, data privacy controls, and policy-aware deployment practices.


Built to address the unique threat landscape of rural EdTech systems, Q-SAFE combines vulnerability scanning, secure authentication, and institutional compliance checks ensuring technology remains both accessible and defensible.

IoT Botnet Detection Using Machine Learning

Completed Research

Investigated anomaly-based detection of IoT botnets by analyzing network traffic patterns, particularly MQTT protocol behavior. Developed machine learning models to identify compromised devices in real time.


Used Python, Scikit-learn, and Pandas to train classifiers on simulated botnet traffic captured via Wireshark. Evaluated performance using precision, recall, and F1-score metrics under varying network conditions.
