1. What is Python?

Python is like a special language that computers understand. It's like learning to talk to your computer in a way it can follow your instructions!

Remember: Python is easy to learn and very powerful!

2. Python Indentation

In Python, spaces at the beginning of a line are very important. They tell Python which code belongs together.

Shortcut: Think of indentation like building blocks - blocks that are indented belong to the block above them!

3. Variables

Variables are like boxes where you can store information. You give each box a name and put something inside.

x = 5
y = "John"
print(x)
print(y)
Memory Trick: Variable = Value (think of the equal sign as an arrow pointing from the name to the value)

4. Global vs Local Variables

Global Variables

These are like boxes that everyone can see and use, anywhere in your code.

Local Variables

These are like secret boxes that only one function can see and use.

5. Python Data Types

Python has different types of data, just like you have different types of toys!

6. Strings

Strings are like words made of letters. You can do fun things with them!

a = "Hello"
print(len(a)) # How many letters?
print(a.upper()) # Make it BIG
print(a.lower()) # Make it small
print(a.replace("H", "J")) # Change letters
String Tricks:
  • len() = length (how long)
  • upper() = SHOUTING
  • lower() = whispering
  • replace() = magic word change

7. Python Collections

Collections are like different kinds of containers for your data!

List

Like a shopping list you can change.

Tuple

Like a treasure chest that's locked.

Set

Like a collection of unique toys.

Dictionary

Like a phone book with names and numbers.

8. If Statements

If statements help your computer make decisions, like choosing what to wear based on the weather!

weather = "sunny"

if weather == "raining":
    print("Take umbrella")
elif weather == "sunny":
    print("Wear sunglasses")
else:
    print("Just go outside!")
Decision Making:
  • if = first choice
  • elif = another choice
  • else = last choice

9. Loops

Loops help you do things over and over again, like counting your toys!

For Loop

Go through each item one by one.

While Loop

Keep going until something changes.

# For loop
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

# While loop
count = 0
while count < 5:
    print(count)
    count = count + 1
Loop Tricks:
  • for = for each item in a list
  • while = while something is true
  • break = stop the loop early
  • continue = skip to the next item

10. Functions

Functions are like magic spells that you can use over and over again!

def add_numbers(num1, num2):
    result = num1 + num2
    return result

# Use the function
answer = add_numbers(2, 3)
print(answer) # Shows 5
Function Formula:
  • def = define (create) a function
  • name = give it a name
  • () = put inputs inside
  • return = give back the answer

11. Python Cheat Codes & Memory Magic

Here are some secret cheat codes to help you write and remember Python easily!

Store Something

name = value

Memory Trick: The equal sign `=` is an arrow pointing from the name to the value.

Make a List

my_list = []

Memory Trick: Square brackets `[]` are an empty shopping list, ready for items.

Make a Dictionary

my_dict = {}

Memory Trick: Curly braces `{}` are a treasure chest with name-tags (keys) for each treasure (value).

Loop Through Items

for item in list:

Memory Trick: "For each `item` that is `in` the `list`, do something..."

Make a Decision

if condition:

Memory Trick: "`if` this is true, then do that. Don't forget the colon `:`!"

Create a Function

def my_func():

Memory Trick: "`def`ine a `my_func`tion. The parentheses `()` are for its ingredients (parameters).

Compare Two Things

a == b

Memory Trick: Use two equals `==` to ask "are they equal?". Use one `=` to give a value.

Get User Input

name = input()

Memory Trick: `input()` is like asking a question and waiting for an answer.

12. Common Errors and Fixes

Even the best programmers make mistakes! Here are some common ones and how to fix them.

Indentation Error

# Wrong
if x > 5:
print("Big")

# Right
if x > 5:
    print("Big")

Name Error

# Wrong
print(my_var)

# Right
my_var = "Hello"
print(my_var)

Syntax Error

# Wrong
if x = 5:

# Right
if x == 5:

# Wrong
print("Hello"

# Right
print("Hello")

Type Error

# Wrong
"5" + 3

# Right
int("5") + 3
# or
"5" + str(3)
Error Fixing Tricks:
  • Check your indentation (spaces at the start)
  • Make sure you defined your variables
  • Use == for comparison, = for assignment
  • Close all your brackets and quotes
  • Make sure your data types match

13. Important Questions for Your Exam

Q1: What is Python?
A: Python is a computer language that helps us give instructions to the computer. It's easy to learn and very powerful!
Q2: What is indentation?
A: Indentation is the spaces at the beginning of a code line. Python uses it to know which code belongs together.
Q3: What is the difference between a global and local variable?
A: A global variable can be used everywhere in your code, but a local variable can only be used inside one function.
Q4: Name some Python data types.
A: Text (str), Numbers (int, float), List, Tuple, Set, Dictionary, Boolean (True/False)
Q5: How do you make a list in Python?
A: Use square brackets: my_list = [1, 2, 3]
Q6: How is a tuple different from a list?
A: A list can be changed (mutable), but a tuple cannot be changed (immutable).
Q7: How do you make a decision in Python?
A: Use if, elif, and else statements.
Q8: What is a loop?
A: A loop is a way to do something over and over again. There are for loops (go through each item) and while loops (keep going while something is true).
Q9: What is a function?
A: A function is a block of code that does a specific job. You can use it many times without writing the code again.
Q10: How do you define a function?
A: Use the def keyword: def my_function():
