Project Requirement Document

SCAR

Smart Classroom Assistant Robot
Target Location: Thrissur, Kerala

Executive Summary

The SCAR is an autonomous classroom robot powered by a Raspberry Pi. It replaces old "remote control" logic with AI-driven autonomy. Designed for the modern Indian classroom, it features face recognition, directional student counting, air quality monitoring, and text-to-speech announcements.

System Architecture

The Brain

Raspberry Pi 4 (4GB RAM)
Running Raspberry Pi OS. Handles all AI processing and logic.

The Eyes

Pi Camera Module 3
High-quality vision for OpenCV-based face recognition.

The Nervous System

Sensors: MQ-135 (Air Quality), DHT22 (Temp & Humidity), Dual IR Sensors (Counting).

The Muscles

Actuation: Pan-Tilt mechanism with MG996R Servos driven by PCA9685.

User Interface

Web Dashboard: Local Flask-based site accessible via classroom Wi-Fi.
